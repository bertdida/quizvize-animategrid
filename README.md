# AnimateGrid

**AnimateGrid** is an interactive 3D wavy grid generator built with Three.js. Easily customize grid parameters, export high-quality PNG snapshots or perfectly loopable MP4 videos, and use for social media, design, or digital art.

## Features

- Real-time 3D wavy grid animation
- Adjustable amplitude, mesh quality, thickness, speed, and colors
- Gradient mode for colorful effects
- Export PNG snapshots (16:9 or 9:16)
- Export 12s loopable MP4 videos (16:9 or 9:16)
- Neobrutalism-inspired UI

## Usage

1. Open [animategrid.online](https://animategrid.online) in your browser.
2. Adjust the controls to customize your grid.
3. Click **Snapshot** to download a PNG, or **Export MP4 Loop** for a 12-second video.

## Deploying

- Hosted on GitHub Pages.
- Custom domain: animategrid.online
- To deploy, push to the `main` branch. The site will update automatically.
